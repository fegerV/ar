# Changelog

Все изменения в проекте Vertex AR будут задокументированы в этом файле.

## [1.0] - 2025-10-29

### Добавлено
- Основной функционал загрузки AR-контента (изображения и видео)
- Генерация NFT-маркеров для AR-отслеживания
- Веб-интерфейс для управления AR-контентом
- Админ-панель для управления контентом
- Система аутентификации и авторизации
- QR-коды для быстрого доступа к AR-контенту
- Интеграция с AR.js и A-Frame для отображения AR
- Счётчик просмотров AR-контента
- Система валидации загружаемых файлов
- Информация о системных ресурсах и использовании хранилища
- Docker-контейнеризация для упрощённого развертывания

### Изменено
- Упрощённая архитектура: SQLite вместо PostgreSQL
- Локальное хранилище вместо MinIO
- Уменьшено количество зависимостей с 15 до 7
- Упрощённое развертывание: 2 контейнера вместо 4

### Исправлено
- Оптимизация загрузки больших файлов
- Сжатие изображений
- Обработка видеофайлов
- Безопасность на уровне API

## [0.5.0] - 2025-10-25

### Добавлено
- Базовый функционал FastAPI приложения
- Система аутентификации
- Загрузка и хранение AR-контента
- Генерация NFT-маркеров
- AR-страница для отображения контента

### Изменено
- Переход от оригинального проекта к упрощённой версии
- Использование SQLite вместо PostgreSQL
- Локальное хранилище вместо MinIO

## [0.1.0] - 2025-10-20

### Добавлено
- Инициализация проекта
- Основная структура приложения
- Базовая аутентификация
- Загрузка файлов
