# План развития Vertex Art AR

## Текущее состояние системы

Vertex Art AR - это веб-приложение для создания и управления AR-контентом, позволяющее пользователям загружать изображения и видео, а затем просматривать их в формате дополненной реальности с использованием технологии AR.js NFT. Приложение использует упрощенную архитектуру с минимальными зависимостями (SQLite, локальное хранилище) и предоставляет функциональность для генерации NFT-маркеров, управления контентом и отслеживания статистики просмотров.

## Краткосрочные цели (6-12 месяцев)

### 1. Улучшение точности отслеживания

**Цель:** Повысить точность и стабильность отслеживания AR-маркеров

**Задачи:**
- Оптимизировать параметры генерации NFT-маркеров (плотность фич, уровни детализации)
- Внедрить сглаживание позиции и ориентации маркера для уменьшения дрожания
- Улучшить обработку изображений при генерации маркеров (улучшение контраста, выделение краев)
- Добавить анализ качества изображения для предсказания пригодности к использованию в AR
- Внедрить альтернативные алгоритмы отслеживания для сложных текстур

**Реализация:**
- Модификация файла `nft_marker_generator.py` для улучшения алгоритмов генерации маркеров
- Добавление новых параметров в `NFTMarkerConfig` для настройки качества
- Внедрение новых эвристик для оценки качества изображений

### 2. Расширение поддержки форматов файлов

**Цель:** Увеличить количество поддерживаемых форматов изображений и видео

**Задачи:**
- Добавить поддержку дополнительных форматов изображений (TIFF, RAW, HEIC)
- Расширить поддержку видеоформатов (MOV, AVI, MKV)
- Внедрить конвертацию файлов в поддерживаемые форматы при загрузке
- Добавить проверку кодеков видео и автоматическую перекодировку при необходимости

**Реализация:**
- Модификация валидатора файлов в `main.py`
- Интеграция с библиотеками для конвертации (например, ffmpeg-python)
- Добавление новых эндпоинтов для обработки различных форматов

### 3. Добавление базовой аналитики

**Цель:** Внедрить систему сбора и анализа данных об использовании приложения

**Задачи:**
- Сбор статистики по просмотрам AR-контента (время просмотра, частота, геолокация)
- Отслеживание успешности/неуспешности отслеживания маркеров
- Сбор данных о типах файлов их характеристиках
- Создание отчетов о производительности и стабильности системы
- Визуализация статистики в админ-панели

**Реализация:**
- Расширение базы данных новыми таблицами для хранения аналитики
- Модификация AR-страницы для отправки аналитических данных
- Добавление новых эндпоинтов API для получения аналитики
- Обновление админ-панели для отображения аналитики

### 4. Улучшение пользовательского интерфейса

**Цель:** Повысить удобство использования приложения

**Задачи:**
- Обновить дизайн админ-панели для лучшей навигации и визуализации контента
- Добавить прогресс-бары для процессов загрузки и генерации
- Внедрить предпросмотр AR-контента перед публикацией
- Улучшить мобильную адаптацию AR-страниц
- Добавить поддержку темной/светлой темы

**Реализация:**
- Обновление HTML-шаблонов в директории `templates`
- Добавление CSS-стилей и JavaScript-функциональности
- Модификация админ-панели для новых UI-элементов
