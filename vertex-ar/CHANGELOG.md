# Журнал изменений Vertex AR (подпроект)

Фиксирует ключевые изменения приложения внутри каталога `vertex-ar/`.

## [1.5.0] - 2024-11-26
- **Добавлен Video Animation Scheduler**: Система автоматического управления видео по расписанию
  - Планирование по времени с start/end datetime
  - Ротация видео (sequential, cyclic, none)
  - Управление статусами (active, inactive, archived)
  - Автоматическая архивация истекших видео
- **Обновлена база данных**: Добавлены колонки scheduling в таблицу videos и таблица video_schedule_history
- **Новые API эндпоинты**: Полный REST API для управления расписанием (/videos/schedule/*)
- **Админ-интерфейс**: Новая панель управления видео расписанием
- **Фоновые задачи**: Автоматическая проверка и активация/деактивация видео
- **Интеграция с уведомлениями**: Автоматические уведомления об изменениях статуса
- **Мониторинг**: Метрики Prometheus и статусы планировщика
- **Документация**: VIDEO_SCHEDULER_FEATURE.md с полным описанием функциональности

## [1.4.0] - 2024-11-21
- Добавлен модуль централизованной валидации (`validators.py`) и middleware для логирования
- Расширены схемы Pydantic и проверки входных данных
- Обновлены маршруты `auth`, `users`, `clients` с аудиторскими логами
- Актуализированы тесты и документация (`PROJECT_DOCS.md`, `README_DEPLOYMENT.md`)

## [1.2.0] - 2024-11-06
- Реализована полноценная система управления пользователями и блокировками
- Добавлены миграции и инициализация базы, rate limiting
- Расширена админ-панель и API для поиска пользователей

## [1.1.0] - 2024-11-05
- Внедрён batch-генератор NFT-маркеров и кэширование анализа
- Добавлены новые эндпоинты для аналитики и управления маркерами
- Оптимизировано хранилище и мониторинг производительности

## [1.0.0] - 2024-10-29
- Первый стабильный релиз: загрузка AR-контента, генерация маркеров, админ-панель
- Docker-конфигурация и скрипты упрощённого деплоя
