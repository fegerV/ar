# Исправление проблемы с обработкой путей резервных копий

## Проблема

В API-методах восстановления и удаления резервных копий возникала ошибка:

```
Access denied: backup must be within backup directory
```

Эта ошибка возникала из-за того, что URL-кодированные параметры пути содержали управляющие символы (например, `\u000b` - вертикальная табуляция и `\u0008` - backspace), которые не удалялись при нормализации пути. В результате проверка безопасности, которая должна убедиться, что запрашиваемый файл находится в разрешённой директории резервных копий, не проходила.

## Причина

Специальные символы, включённые в URL-кодированные параметры, не были должным образом очищены перед проверкой пути. Это происходило при передаче параметров через URL, где символы новой строки, табуляции и другие управляющие символы могли искажать воспринимаемое местоположение файла.

## Решение

Внесены изменения в файл `vertex-ar/app/api/backups.py`:

1. В методе `restore_backup` (строки 209-218):
   - Добавлена очистка управляющих символов из пути перед его обработкой
   - Путь теперь очищается от символов `\u000b`, `\u0008`, `\n`, `\r`, `\t`

2. В методе `delete_backup` (строки 345-354):
   - Добавлена аналогичная очистка управляющих символов
   - Обработка каждого пути из списка (для полных резервных копий)

## Код изменений

```python
# В методе restore_backup:
clean_backup_path = request.backup_path.strip().replace('\u000b', '').replace('\u0008', '').replace('\n', '').replace('\r', '').replace('\t', '')
backup_path = Path(clean_backup_path)

# В методе delete_backup:
clean_path_str = path_str.strip().replace('\u000b', '').replace('\u0008', '').replace('\n', '').replace('\r', '').replace('\t', '')
backup_file = Path(clean_path_str)
```

## Результат

- Устранена ошибка "Access denied: backup must be within backup directory"
- Улучшена безопасность обработки путей к резервным копиям
- Повысилась надёжность операций восстановления и удаления резервных копий
- Добавлен тест `test_files/test_backup_path_fix.py` для проверки корректной обработки путей с управляющими символами
