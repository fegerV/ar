# Vertex AR

## Описание проекта

Vertex AR - это веб-приложение, которое позволяет "оживлять" портреты с помощью технологии дополненной реальности (AR). Пользователь может загрузить изображение портрета, и система создаст AR-контент, который можно просматривать через камеру смартфона.

## Уникальная функция - "Оживление" портретов

Наша киллер-фича - возможность оживлять статичные портреты с помощью AR-анимации. В отличие от конкурентов, которые предлагают печать портретов на различных материалах, мы добавляем интерактивный элемент: при наведении камеры смартфона на напечатанный портрет пользователь видит анимированный контент с морганием, улыбкой и другими анимациями.

## Особенности

- Загрузка изображений и создание AR-контента
- Поддержка NFT-маркеров для точного распознавания
- Анимации: моргание, улыбка, поворот головы, подмигивание
- Генерация QR-кодов для быстрого доступа к AR-контенту
- Админ-панель для управления контентом
- Статистика просмотров

## Технологии

- FastAPI - веб-фреймворк
- A-Frame и AR.js - для AR-функциональности
- Anime.js - для анимаций
- SQLite - база данных
- MinIO - хранение файлов

## Установка

1. Клонируйте репозиторий
2. Установите зависимости: `pip install -r requirements.txt`
3. Запустите приложение: `python main.py`

## Использование

1. Зарегистрируйтесь и залогиньтесь как администратор
2. Загрузите изображение портрета и видео
3. Система сгенерирует AR-контент и QR-код
4. Напечатайте портрет и используйте камеру смартфона для просмотра анимации

## API endpoints

- `POST /auth/register` - регистрация пользователя
- `POST /auth/login` - аутентификация
- `POST /ar/upload` - загрузка AR-контента
- `GET /ar/{content_id}` - просмотр AR-контента
- `GET /admin` - админ-панель
